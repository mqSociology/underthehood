Some useful resources for managing content:

  • Markdown cheat sheet there are many more of these that can be found through google.

  • {RMarkdown cheat sheet] (https://www.rstudio.com/resources/cheatsheets/#rmarkdown) again there are many more of these

  • Academic theme documentation I believe this was geared towards Markdown users and not RMarkdown users

  • I encourage you to look at the Stackoverflow forums when googling a problem

  • W3Schools has a pretty good list of CSS variables if you are looking to modify your content with some inline CSS styling (not normally recommended but useful for little things like changing the colour of text)
